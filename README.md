Hora de Correr é um aplicativo de teste criado para uma avaliação escolar, que consiste no desenvolvimento de um pequeno jogo utilizando conceitos lógicos para a comparação de grandezas de velocidade. O aplicativo possui uma interface simples e intuitiva, além de um sistema de login básico com e-mail e senha.

A ideia foi criar um aplicativo mais descontraído e divertido, já que a proposta do projeto era livre. O sistema utiliza o Java como sua principal linguagem de desenvolvimento.
